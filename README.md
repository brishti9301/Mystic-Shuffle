# Mystic Shuffle 🔮

Mystic Shuffle is an interactive tarot reading web application built with HTML, CSS, and JavaScript.

## Features

- Ask a question
- Shuffle the tarot deck
- Choose a card
- Personalized tarot interpretations
- Interactive UI
